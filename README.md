# PROJECT 1 📌
## 🤖 Smart ChatBot

A rule-based chatbot developed in Python that interacts with users through predefined responses and conversational commands.

### 📌 Overview

Simple ChatBot is a beginner-friendly Python project that simulates a basic conversation between a user and a chatbot. The chatbot uses conditional statements and predefined rules to respond to user inputs.
This project is ideal for learning core Python programming concepts and understanding how simple chatbots work before exploring advanced AI-based systems.

---

### ✨ Features

* 👋 Personalized Greetings
* 🧑 User Name Recognition
* 💬 Interactive Conversations
* 😂 Joke Generator
* 📖 Help Menu
* 🙏 Thank You Responses
* 🔄 Continuous Chat Loop
* ❌ Invalid Input Handling
* 🚪 Exit Commands (`bye`, `exit`, `quit`)

---

### 🛠 Technologies Used

* Python 3.x
* Functions
* Loops
* Conditional Statements
* String Manipulation
* User Input/Output

---

## 💬 Available Commands

| User Input        | Bot Response          |
| ----------------- | --------------------- |
| hello / hi / hey  | Greeting Message      |
| how are you       | Status Response       |
| what's your name  | Bot Introduction      |
| tell me a joke    | Random Joke           |
| help              | Available Commands    |
| thank you         | Appreciation Response |
| bye / exit / quit | Ends Conversation     |

---

```text
Bot: What's your name?
```

# PROJECT 2 📌
## Hangman Deluxe

A feature-rich Hangman game developed in Python that combines classic gameplay with score tracking, difficulty levels, and persistent high-score storage.

### 📌 Overview

Hangman Deluxe is a word-guessing game where the player attempts to discover a hidden word by guessing one letter at a time. The player wins by revealing the entire word before the hangman drawing is completed.
This project demonstrates fundamental and intermediate Python programming concepts including file handling, functions, loops, conditional statements, and randomization.

---

### ✨ Features

* Multiple Difficulty Levels
  - Easy
  - Medium
  - Hard
* Random Word Selection
* ASCII Hangman Graphics
* Score Tracking System
* High Score Storage using File Handling
* Replay Functionality
* Input Validation
* User-Friendly Interface

---

### 🛠 Technologies Used

* Python 3.x
* Random Module
* File Handling
* Functions
* Loops
* Conditional Statements

---

### 🎮 Game Rules

1. Select a difficulty level.
2. A random word is chosen.
3. Guess one letter at a time.
4. Correct guesses reveal hidden letters.
5. Incorrect guesses draw parts of the hangman.
6. The game ends when:
   - The word is guessed correctly, or
   - The maximum number of wrong guesses is reached.

---

### 🏆 Scoring System

| Action         | Score               |
| -------------- | ------------------- |
| Win a Game     | +10 Points          |
| Lose a Game    | 0 Points            |
| New High Score | Saved Automatically |
